# Workshop เสริมที่ 1: การเรียงลำดับข้อมูลในรายงาน (Custom Sorting)

## สถานการณ์จำลอง

ท่านได้รับ Feedback จากหัวหน้าว่าอยากเรียงลำดับหน่วยปฏิบัติตามโครงสร้างหน่วยงาน คือ ส่วนกลาง, สจป. 1-13, สจป. สาขาทุกสาขา

## วัตถุประสงค์

เพื่อเรียงลำดับหน่วยปฏิบัติตามโครงสร้างหน่วยงาน

## สร้างแผ่นงานอ้างอิง

1. สร้างแผ่นงานใหม่ใน Google Sheets ชื่อ **หน่วยปฏิบัติ**
2. สร้างคอลัมน์ 2 คอลัมน์:

    - **ลำดับหน่วยปฏิบัติ** (สำหรับใส่ตัวเลขเรียงลำดับ)
    - **ชื่อหน่วยปฏิบัติ** (สำหรับใส่ชื่อหน่วยงาน)

3. เพิ่มข้อมูลตามลำดับที่ต้องการ โดยเริ่มจากส่วนกลาง, สจป. 1-13 และตามด้วยสาขาทุกสาขา
4. **สำคัญ**: ข้อมูลในคอลัมน์ *ชื่อหน่วยปฏิบัติ* จะต้องตรงกับข้อมูลในคอลัมน์ *หน่วยปฏิบัติ* ของแผ่นงาน **Master_Data** เพื่อให้สามารถเชื่อมโยงข้อมูลกันได้

## การผสมข้อมูล (Data Blending) ใน Looker Studio

1. เพิ่มแหล่งข้อมูลแผ่นงาน **หน่วยปฏิบัติ** เข้ามาในรายงาน
2. ทำการผสมข้อมูล (Blend Data) ระหว่าง **Master_Data** และ **หน่วยปฏิบัติ**
    - ใช้คอลัมน์ **ชื่อหน่วยปฏิบัติ** เป็น Join Key
3. ดึงคอลัมน์ **ลำดับหน่วยปฏิบัติ** เข้ามาใน Data Source ที่ผสมแล้ว

## การจัดเรียงข้อมูลในแผนภูมิ

1. คลิกที่แผนภูมิที่ต้องการจัดเรียง
2. ในส่วนของคุณสมบัติ (Properties) ไปที่หัวข้อ **Sort** (จัดเรียง)
3. เลือกฟิลด์ **ลำดับหน่วยปฏิบัติ**
4. กำหนดให้เรียงจากน้อยไปมาก (Ascending)
