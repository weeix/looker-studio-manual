# Workshop 6: การตั้งค่าการแสดงผล (Layout Settings)

## สถานการณ์จำลองที่

ผู้บังคับบัญชาชมรายงานผ่านมือถือบ่อย ต้องการตัวหนังสือใหญ่ กดง่าย และไม่ต้องซูมเข้าออกบ่อย ๆ

## วัตถุประสงค์

1. ปรับ Layout ของรายงานให้เหมาะกับการแสดงผลบนหน้าจอมือถือ
2. ทำให้ผู้ใช้เลื่อนดูกราฟได้สะดวกในแนวตั้ง

## การตั้งค่า Layout ให้เหมาะกับมือถือ

1. คลิกพื้นที่ว่างของรายงาน  
2. ไปที่เมนู **Theme and layout**  
3. เปิดแท็บ **Layout** แล้วเลือก **Display Mode** เป็น **Fit to width**

ผลลัพธ์: รายงานขยายเต็มความกว้างหน้าจอมือถือ อ่านง่ายไม่ต้องซูม

**Pro Tip:** จัดกราฟเรียงในแนวตั้งเพื่อให้เลื่อนดูสะดวกบนมือถือ
